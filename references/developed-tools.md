# Herramientas ya desarrolladas que solo requiren configuración para el caso de uso IA

## Autoría
- **Oxygen XML Editor** es el único editor XML comercial instalado en local que permite, sin crear código, indexar a su RAG un repositorio de documentos XML de referencia y la documentación técnica legacy volcada en un CCMS o CSDB. Además posee un MCP Client al que se le podría conectar un MCP Server con información de Source Data a la que acceder. Internamente, Oxygen también dispone de un MCP Server con funciones para la edición (agente IA Positron). Coste: 1.200 € licencia perpetua. Configurable con cualquier LLM mediante API Key. Dispone de SDK para desarrolladores.
- **Visual Studio Code** podría servir como XML editor básico con LLMs embebidos (Codex, Claude Code, Gemini) para proyectos de menor presupuesto 
- **Xignal** es una CSDB para S1000D que tiene conector a LLMs y puede conectarse a PLMs para recuperar datos logísticos. Es exclusivo para S1000D.
- **Heretto CCMS** es un CCMS para DITA, tiene integrado *Etto* como asistente IA para trabajar en DITA en su editor XML
- **IXIA CCMS** es un CCMS para DITA, tiene *Syndicate* para gestión IA de publicaciones
- **RWS Tridion** es un CCMS para DITA, tiene MCP conectado a Claude para ediciones y control de publicaciones desde el chat (también dispone de editor XML integrado)
- **ST4** es un CCMS para DITA, tiene asistente IA (funciones por confirmar)
- **HyperSTE (Etteplan)** es un pluggin para proyectos con requisito STE (Standard Technical English), usa tecnología IA (pendiente de evaluación de la herramienta)
